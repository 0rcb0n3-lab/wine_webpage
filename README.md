# Магазин авторского вина «Новое русское вино»

Веб-сайт с каталогом винодельческой продукции.

---

## Быстрый запуск

### 1. Подготовка окружения

Установите необходимые пакеты:

```bash
pip install -r requirepment.txt
```

- Проверьте наличие ```wines.xlsx``` в папке проекта, если его нет, создайте и заполните по инструкции ниже

### 2. Запуск локального сервера

Программа поддерживает конфигурацию источника данных через аргументы командной строки:

- **Стандартный запуск** (использует встроенный файл `wine.xlsx`):

    ```bash
    python3 main.py
    ```

- **Альтернативный запуск**

 (с указанием альтернативной таблицы):

```bash
python3 main.py --file path/to/catalog.xlsx
```

(c конкретного листа по его имени)

```bash
python3 main.py --sheet "Лист№"
```

*Скрипт проигнорирует порядок и найдет в таблице лист с точным названием "Лист№".*

(комбинированный запуск с другим файлом и листом)

```bash
python3 main.py --file smthing.xlsx --sheet "РандомныйЛист"
```

После запуска сайт будет доступен по адресу: ```http://127.0.0.1:8000```.

---

## Спецификация входных данных

Для корректной работы системы используемый `.xlsx` файл должен содержать рабочий лист **`Лист1`** со следующей структурой столбцов:

| Столбец | Тип данных | Описание |
| :--- | :--- | :--- |
| **Категория** | Строка | Название раздела (например, *Красные вина*) |
| **Название** | Строка | Наименование позиции |
| **Сорт** | Строка | Сорт винограда|
| **Цена** | Число | Стоимость товара |
| **Картинка** | Строка | Имя файла изображения из папки медиа-ресурсов |
| **Акция** | Строка | Маркер специального предложения|

*Примечание: сборка HTML-страницы происходит единожды в момент инициализации сервера. При внесении изменений в Excel-таблицу требуется перезапуск скрипта `main.py`.*
